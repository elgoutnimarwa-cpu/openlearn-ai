# openlearn-ai
OpenLearn AI is an open-source educational platform that provides AI-powered tutoring, personalized study plans, quiz generation, and collaborative learning tools for students worldwide.
# OpenLearn AI

![License](https://img.shields.io/badge/license-MIT-blue.svg)

OpenLearn AI is an open-source educational platform designed to make high-quality learning accessible to everyone through AI-powered assistance.

## Vision

To democratize education by providing intelligent learning tools that help students learn faster, retain more information, and achieve their academic goals.

## Features

- AI Study Assistant
- Personalized Learning Paths
- Quiz Generator
- Learning Analytics
- Progress Tracking
- Community Learning Spaces
- Open Educational Resources

## Tech Stack

### Backend
- Python
- FastAPI
- PostgreSQL

### Frontend
- React
- TypeScript
- Tailwind CSS

### AI
- OpenAI API
- LangChain

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/openlearn-ai.git

cd openlearn-ai

pip install -r requirements.txt

uvicorn backend.main:app --reload
```
